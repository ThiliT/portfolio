# Model Attributions

Add one entry per model you include in `public/models/`.

Template:

```
Name: <Model Name>
Source: <URL>
Author: <Author Name + profile URL>
License: <License name + URL>
Filename: /public/models/<file>
Notes: <Optimization/edit notes>
```

Example:

```
Name: Laptop (Low Poly)
Source: https://sketchfab.com/...
Author: Jane Doe (https://sketchfab.com/janedoe)
License: CC BY 4.0 (https://creativecommons.org/licenses/by/4.0/)
Filename: /public/models/laptop.glb
Notes: Draco compressed with gltf-pipeline
```
